Let's Watch

Native Android App developed in Kotlin

This app was nativly written in Kotlin as a project for my uni. It's main purpose is to connect two users and suggest them movies. They decide if the like the movie, don't like or if they have watched. If both users liked the same movie it add that movie on a to watch list. Once the have watch it they can remove it. As a database it uses Firebase and all of it's capabilities. This project sadly doesn't use MVVM architecture because it is more of an playground for testing android development and also because it was an oversight that wasn't noticed until later into the project

PID regulation of wooden stove

Arduino - Hardware

Let's just say I got tired od old analog/mechanical way of controlling the airflow for the furnace on wood. So I decided to add PID regulation onto it. It has 2 analog PID regulators controlling air intake and exhaust. It was also coupled with a potentiometer for controlling and setting the temperature. That temperature is controled with thermocouples controlling water temperature inside the furnace, outgoing pipe and return pipe for the central heating system. With that I accomplished precise control over the temperature and prevented potential overheating which happened a few times with the old mechanical way of controlling the furnace. Also since the pipes for central heating are going aoutside for a short size, I added an outside temperature sensor which when it is bellow 3 °C activated the pump for a short duration even if the furnace isn't running so it removes the possibility of the water freezing inside the pipes.

Temperature display

VHDL - I2C communication

This project was done for the uni on one of theirs Nexys A7 boards. The main goal was to establish I2C communication with the onboard temperature sensor and request data. After that, the data was processed and the output was displayed on the onboard 7 segment displays. It was measuring with three decimal places with the resolution of 0.125°C. The whole programming was done in the VHDL hardware programming language which realy puts you into the hardware mindset and requires full concentration.
