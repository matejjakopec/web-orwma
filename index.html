Let's Watch

Native Android - Kotlin

The idea of this app is to make the activity of what to watch with someone a lot simpler. It uses API with tousands of movies and suggests them to you and if both of you like that movie, if promotes it to you (something like Tinder, but for movies).

PID regulation of furnace on wood

Arduino - Hardware

Simply put I was fed up with mechanical "control" on the current furnace and decided to add PID regulation with motors on input and exhaust of the furnace

Temperature display

VHDL - I2C communication

Project that focused on more hardware level. Uses I2C communication with the onboard temperature chip and displays the temperature on 7 segment display
